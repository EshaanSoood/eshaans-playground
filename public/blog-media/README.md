# Blog Media

Use this folder for assets that belong only to the blog.

Suggested structure:

- `images/` for post images, galleries, artwork, diagrams
- `audio/` for clips, demos, stems, or short story-linked sound bites

Reference them in posts with paths like:

- `/blog-media/images/my-photo.jpg`
- `/blog-media/audio/my-clip.mp3`

These files are separate from the main site's static assets in `public/` at the workspace root.
